Informática II

Librerías

Standard Input/Output:

#include <stdio.h>

printf: Print Formatted

printf();

Imprime texto formateado en la salida estándar.

scanf: Scan Formatted

scanf();

Lee datos formateados desde la entrada estándar.

fflush: File Flush

fflush(stdin);

Vacía el buffer de un flujo.

gets: Get String

gets();

Lee una línea desde stdin (obsoleta e insegura). Usar fgets().

fgets: File Get String

fgets();

Lee una línea de un archivo o de stdin con límite de tamaño.

getchar: Get Character

getchar();

Lee un carácter desde stdin

fopen: File Open

fopen();

Abre un archivo.

fclose: File Close

fclose();

Cierra un archivo.

fputc: File Put Character

fputc();

Escribe un carácter en un archivo.

fgetc: File Get Character

fgetc();

Lee un carácter de un archivo.

feof: File End Of File

feof();

Indica si se llegó al final del archivo.

ferror: File Error

ferror();

Detecta errores en un flujo de archivo.

putchar: Put Character

putchar();

Escribe un carácter en stdout.

puts: Put String

puts();

Imprime una cadena y agrega salto de línea.

fputs: File Put String

fputs();

Escribe una cadena en un archivo.

fprintf: File Print Formatted

fprintf();

Imprime texto formateado en un archivo.

fscanf: File Scan Formatted

fscanf();

Lee datos formateados desde un archivo.

sscanf: String Scan Formatted

sscanf();

Lee datos formateados desde una cadena.

fwrite: File Write

fwrite();

Escribe bloques de datos binarios en un archivo.

fread: File Read

fread();

Lee bloques de datos binarios de un archivo.

ftell: File Tell

ftell();

Devuelve la posición actual en el archivo.

fseek:File Seek

fseek();

Mueve el puntero dentro de un archivo.

rewind: Re-Wind

rewind();

Vuelve al inicio del archivo.

remove: Remove File

remove();

Elimina un archivo del sistema.

Console Input/Output

#include <conio.h>

getche: Get Character Echo

getche();

Lee un carácter y lo muestra en pantalla.

clrscr: Clear Screen

clrscr();

Limpia la pantalla de la consola.

getch: Get Character

getch();

Lee un carácter sin mostrarlo.

Standard Definitions

#include <stddef.h>

sizeof(): Size Of

sizeof();

Operador que devuelve el tamaño en bytes de un tipo o variable.

Standard Library

#include <stdlib.h>

system(): System Command

system();

Ejecuta un comando del sistema operativo.

rand: Random

rand();

Genera un número aleatorio.

exit: Exit Program

exit();

Termina el programa inmediatamente.

malloc: Memory allocation

malloc();

Reserva memoria dinámica.

calloc: Contiguous Allocation

calloc();

Reserva memoria dinámica e inicializa en cero.

realloc: Re-Allocation

realloc();

Cambia el tamaño de un bloque de memoria.

free: Free Memory

free();

Libera memoria dinámica.

Mathematics

#include <math.h>

max: Maximum

max();

Devuelve el mayor de dos valores (No estándar en C).

pow: Power

pow();

Calcula una potencia.

sqrt: Square Root

sqrt();

Calcula la raíz cuadrada.

atan2: Arc Tangent (2 arguments)

atan2();

Calcula la arcotangente considerando cuadrante.

cos: Cosine

cos();

Calcula el coseno.

sin: Sine

sin();

Calcula el seno.

String handling

#include <string.h>

strcspn: String Complement Span

strcspn();

Calcula longitud hasta encontrar caracteres de otro string.

strcmp: String Compare

strcmp();

Compara dos cadenas.

strcpy: String Copy

strcpy();

Copia una cadena en otra.

Character Type

#include <ctype.h>

toupper: To Upper Case

toupper();

Convierte un carácter a mayúsculas.

Entorno de desarrollo

MinGW64 + Visual Studio Code

Obtener MinGW64 que contiene GCC y GDB, descargarlo, descomprimirlo en la unidad C en la carpeta mingw64

$url="https://github.com/niXman/mingw-builds-binaries/releases/download/15.2.0-rt_v13-rev1/x86_64-15.2.0-release-posix-seh-ucrt-rt_v13-rev1.7z"; $tmp="$env:TEMP\mingw.7z"; $dir="$env:TEMP\mingw_extract"; Invoke-WebRequest $url -OutFile $tmp; New-Item -ItemType Directory -Force -Path $dir | Out-Null; tar -xf $tmp -C $dir; Move-Item "$dir\mingw64" "C:\mingw64"

Tras terminar y ver nuevamente que podemos introducir comandos, agragamos gcc y gdb a las variables de entorno

setx PATH "$env:PATH;C:\mingw64\bin"

Cerrar la terminal y abrir una nueva para comprobar que se agrego:

gcc --version

Instalar Microsoft Visual Studio Code y las extensiones necesarias para C

Abrimos otra terminal de powershell e ingresamos la siguiente linea:

winget install Microsoft.VisualStudioCode ; & "$env:LOCALAPPDATA\Programs\Microsoft VS Code\bin\code.cmd" --install-extension ms-vscode.cpptools ms-vscode.cpptools-extension-pack

Abrimos VS Code
